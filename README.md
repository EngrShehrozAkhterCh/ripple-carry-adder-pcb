Just completed my first custom PCB — a 4-bit Ripple Carry Adder, fully built from scratch.
Here's the complete flow I went through for my Digital Logic (EE-272) project at UET Lahore:
🔷 Schematic — Designed in EasyEDA using CD74HC283E adder IC, 8 push-button inputs, LM7805 voltage regulator, and LED outputs for sum bits + carry-out
🔷 PCB Layout — Manually routed all traces on a 2-layer board and exported Gerbers
🔷 Simulation — Verified logic in Proteus before fabrication
🔷 Manufacturing — Had the board fabricated and soldered every component by hand
The result: press any input combination, LEDs show the binary sum and carry-out in real time.
What this taught me: schematic capture is the easy part. Trace routing, component placement, and getting your solder joints right — that's where real understanding happens.
