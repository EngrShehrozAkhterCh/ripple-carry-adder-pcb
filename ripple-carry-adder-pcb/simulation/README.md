# Simulation

Proteus 8 simulation file for the 4-bit Ripple Carry Adder.

## Requirements
- Proteus 8 or later
- Uses standard 74HC and LED libraries (included in default Proteus installation)

## File
- `pcb adder.pdsprj` — main Proteus project file

## How to Run
1. Open Proteus 8
2. File → Open Project → select `pcb adder.pdsprj`
3. Press the Play button
4. Toggle the input switches to test different A and B values
5. Observe sum output on LEDs (S0–S3) and carry-out (Cout)
